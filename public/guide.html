Comment utiliser l'éditeur
How to use the editor
Importez une image, annotez-la avec les outils, puis exportez en haute résolution.
Import an image, annotate it with the tools, then export in high resolution.
Étape 01
Step 01
📂

Importer une image

Import an image

Chargez une ou plusieurs images depuis votre ordinateur, capturez via webcam, collez une capture d'écran ou glissez-déposez des fichiers. Le multi-import permet de gérer un lot complet en une session.

Load one or more images from your computer, capture via webcam, paste a screenshot or drag & drop files. Multi-import lets you handle an entire batch in one session.

Fichier local Local file Webcam Webcam Capture d'écran Screenshot
Étape 02
Step 02
✏️

Annoter et éditer

Annotate & edit

Utilisez les 11 outils de la barre latérale gauche. Ajustez la luminosité, le contraste, la saturation et la netteté depuis le panneau droit. Utilisez les outils Lasso pour la conformité RGPD.

Use the 11 tools in the left sidebar. Adjust brightness, contrast, saturation and sharpness from the right panel. Use Lasso tools for GDPR compliance.

Exporter en haute résolution

Export in high resolution

Enregistrez en PNG via Ctrl+S. Choisissez la qualité (HQ/MQ/BQ) et exportez l'image actuelle ou tout le lot en une seule session. Copiez aussi directement dans le presse-papier.

Save as PNG via Ctrl+S. Choose quality (HQ/MQ/BQ) and export the current image or the whole batch in one session. Or copy directly to the clipboard.

Anonymise sensitive areas
Les outils Lasso permettent de tracer librement un contour autour de visages, plaques ou données personnelles pour les masquer ou les effacer.
Lasso tools let you freely trace a contour around faces, plates or personal data to blur or erase them.
🌀 Lasso Flou (touche 0)
🌀 Blur Lasso (key 0)

Tracez un lasso autour de la zone à flouter. Au relâchement, l'éditeur applique le traitement sélectionné sur la zone délimitée.

Draw a lasso around the area to blur. On release, the editor applies the selected treatment to the delimited area.

Paramètres outil
Tool parameters
Mode Mode Flou gaussien ou Pixellisation Gaussian blur or Pixelate
Intensité Intensity 1 (léger) → 20 (très fort) 1 (light) → 20 (very strong)
🧹 Lasso Effacer (touche E)
🧹 Erase Lasso (key E)

Tracez un lasso pour effacer pixel à pixel la zone délimitée — transparence complète, irréversible dans la session en cours.

Draw a lasso to erase pixel by pixel the delimited area — complete transparency, irreversible within the current session.

⚠️ À utiliser avec précaution — les pixels effacés ne peuvent pas être récupérés.

⚠️ Use with care — erased pixels cannot be recovered.

💡 Pendant le tracé du lasso, les objets existants sont temporairement verrouillés pour éviter les déplacements accidentels. Ils sont restaurés automatiquement dès la fin du tracé. While drawing the lasso, existing objects are temporarily locked to prevent accidental moves. They are automatically restored when the lasso is released.
