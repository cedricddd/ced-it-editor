Importer une image

Import an image

Chargez une ou plusieurs images depuis votre ordinateur, capturez via webcam ou collez une capture d'écran. Le multi-import permet de gérer un lot complet en une session.

Load one or more images from your computer, capture via webcam, or paste a screenshot. Multi-import lets you handle an entire batch in one session.

Export in high resolution
